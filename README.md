# 📊 Student Performance Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes student academic performance based on gender, school, study time, and attendance.

## 🛠 Tools Used
- Power BI
- DAX
- Data Cleaning (Power Query)

## 📈 Key Insights
- Overall student performance analysis
- Pass percentage calculation
- Study time impact on marks
- Attendance vs performance relationship
